# Opera Add-ons Store Submission Materials

This directory contains all materials needed for submitting Blueprint MCP to the Opera Add-ons store.

## Required Materials

### Text Content

1. **SUMMARY.txt** (✅ Ready)
   - Short description for the listing
   - Character limit: ~250 characters
   - Current: 200 characters

2. **DESCRIPTION.txt** (✅ Ready)
   - Full description of the extension
   - Formatted for Opera Add-ons store
   - Adapted from Chrome version with Opera-specific changes

### Images

Opera Add-ons requires:

1. **Icon** (✅ Ready)
   - 64x64px: `icon-64.png` (created from extension icon)

2. **Screenshots** - Can use Chrome screenshots
   - Recommended: 1280x800px or 1920x1080px
   - At least 1 screenshot required
   - We have: `screenshot_free.png` and `screenshot_pro.png` from Chrome store

3. **Promotional Image** (✅ Ready)
   - 300x188px: `promo-300x188.png` (for Opera featured listing)
   - Contains extension name, logo, and tagline
   - Adapted from Chrome marquee tile

## Submission Checklist

- [x] Extension package (blueprint-mcp-opera-v1.9.4.zip)
- [x] Summary text
- [x] Description text
- [x] Icon 64x64 (icon-64.png)
- [x] Promotional image 300x188 (promo-300x188.png)
- [ ] Screenshots (use Chrome versions)

## Store Listing URL

After submission: https://addons.opera.com/extensions/details/blueprint-mcp/

## Notes

- Opera is Chromium-based, so the extension and functionality are identical to Chrome
- Screenshots from Chrome store can be reused (they show the same UI/functionality)
- All promotional images from Chrome store are compatible

Images can be copied from:

- Promotional: `../chrome/*.png`
